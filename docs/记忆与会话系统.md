# 记忆与会话系统

> BananaBot 的记忆与会话管理模块设计文档

---

## 一、核心概念

| 概念 | 用途 | 存储位置 | 生命周期 |
|------|------|----------|----------|
| **Session** | 短期对话历史 | `~/.bananabot/sessions/` | 会话期间 |
| **Memory** | 长期跨会话知识 | `~/.bananabot/projects/<hash>/memory/` | 永久 |

### 1.1 Session（会话）

**设计理念**：模拟人类对话的"短期记忆"，记录当前对话的完整往来。

- 每一次对话（无论多长）是一个 Session
- Session 内记录所有 user/assistant/tool 消息
- 切换 `/session <name>` 会创建/切换到新会话，旧会话保留

### 1.2 Memory（记忆）

**设计理念**：模拟人类的"长期记忆"，跨会话记住重要知识。

- 不随会话结束而消失
- 按工作区（workspace）隔离，不同项目有不同记忆
- 需要显式触发保存（`/memory save`）

## 二、目录结构

```
~/.bananabot/                              # 全局目录
├── BANANA.md                             # 全局指令（所有项目生效）

│   ├── cli:default.jsonl
│   ├── cli:20260414120000.jsonl
│   └── mychat.jsonl
└── projects/                            # 按项目隔离
    └── <cwd-hash>/                      # 工作区 MD5 hash 前 8 位
        └── memory/
            ├── MEMORY.md               # 记忆入口
            └── *.md                     # 主题记忆
```

## 三、模块说明

### 3.1 memory.py — 记忆管理系统

**职责**：管理长期记忆的存储和加载

**核心类**：`MemoryStore`

```python
class MemoryStore:
    def __init__(self, workspace: Path, global_dir: Path)
    def get_memory_context(self) -> str | None      # 获取记忆（前 200 行）
    def save_memory(self, content: str)              # 保存记忆
    def add_note(self, topic: str, content: str)    # 添加主题记忆
    def list_notes(self) -> list[str]               # 列出主题记忆
    @staticmethod
    def find_banana_md(workspace: Path) -> Path | None  # 查找 BANANA.md
```

**项目隔离原理**：
```python
project_hash = hashlib.md5(str(workspace.resolve()).encode()).hexdigest()[:8]
memory_dir = global_dir / "projects" / project_hash / "memory"
```

**BANANA.md 查找顺序**（向上遍历）：
```
./BANANA.md
./.bananabot/BANANA.md
../BANANA.md
../.bananabot/BANANA.md
...
```

---

### 3.2 session.py — 会话管理系统

**职责**：管理会话的创建、保存、加载

**核心类**：`Session` 和 `SessionManager`

**Session**：
```python
class Session:
    def __init__(self, key: str, messages: list | None = None, path: Path | None = None)
    def add(self, role: str, content: str, **kwargs)     # 添加消息
    def history(self) -> list                           # 获取历史副本
    def clear(self)                                     # 清空历史
    def save(self)                                      # 保存到 JSONL
    @classmethod
    def load(cls, path: Path) -> "Session"              # 从 JSONL 加载
```

**SessionManager**：
```python
class SessionManager:
    def __init__(self, sessions_dir: Path)
    def get_or_create(self, key: str) -> Session       # 获取或创建会话
    def list_sessions(self) -> list[str]                # 列出所有会话
    def delete(self, key: str)                          # 删除会话
```

**存储格式**：JSONL（每行一个 JSON 对象）
```json
{"role": "user", "content": "你好", "timestamp": "2026-04-14T12:00:00.000000"}
{"role": "assistant", "content": "干啥", "timestamp": "2026-04-14T12:00:01.000000"}
```

### 3.3 context.py — 上下文构建器

**职责**：构建发送给 LLM 的完整消息列表

**消息加载顺序**：
```
1. 全局 BANANA.md（~/.bananabot/BANANA.md）
2. 项目 BANANA.md（从 workspace 向上查找）
3. Memory 记忆（~/.bananabot/projects/<hash>/memory/MEMORY.md）
4. 系统提示词模板
```

**输出格式**：
```python
[
    {"role": "system", "content": "<完整 system prompt>"},
    {"role": "user", "content": "<历史消息>"},      # 来自 session.history()
    {"role": "assistant", "content": "<历史消息>"},  # 来自 session.history()
    ...
    {"role": "user", "content": "<当前输入>"}
]
```

## 四、用户交互命令

### 4.1 会话相关命令

| 命令 | 功能 |
|------|------|
| `/new` | 创建新会话（带时间戳） |
| `/session <name>` | 切换到指定会话 |
| `/sessions` | 列出所有会话 |
| `/clear` | 清空当前会话历史 |
| `/status` | 显示当前状态 |

### 4.2 记忆相关命令

| 命令 | 功能 |
|------|------|
| `/memory` | 查看当前记忆 |
| `/memory save` | 总结会话历史并追加到记忆（自动调用 LLM 总结） |
| `/banana` | 查看全局/项目指令 |

### 4.3 命令示例

```bash
# 会话操作
BananaBot: /new                      # 开启新会话
BananaBot: /session myproject        # 切换到 myproject 会话
BananaBot: /sessions                 # 查看有哪些会话
BananaBot: /clear                    # 清空当前会话

# 记忆操作
BananaBot: /memory                   # 查看当前记忆
BananaBot: /memory save              # 总结当前会话历史，自动追加到记忆
BananaBot: /banana                   # 查看全局/项目指令
```

### 4.4 /memory save 工作流程

```
1. 读取现有记忆（MEMORY.md）
2. 获取当前会话历史（JSONL）
3. 调用 LLM 总结：哪些重要信息需要永久记住
4. 追加总结到 MEMORY.md
5. 如果超过 200 行，再次调用 LLM 精简
```

## 五、配置参数

### 5.1 Config 中的相关配置

```python
# 工作区 = 当前命令执行目录
self.workspace = Path.cwd().resolve()

# 全局目录 ~/.bananabot/
self.global_dir = Path.home() / ".bananabot"

# 会话存储目录
self.sessions_dir = self.global_dir / "sessions"

# 项目记忆目录
self.memory_dir = self.global_dir / "projects" / self._hash_cwd() / "memory"
```

### 5.2 环境变量

| 变量 | 默认值 | 说明 |
|------|--------|------|
| `BASE_URL` | `https://api.openai.com/v1` | LLM API 地址 |
| `LLM` | `gpt-4o` | 模型名称 |
| `API_KEY` | `` | API 密钥 |
| `MAX_ITERATIONS` | `20` | 最大迭代次数 |
